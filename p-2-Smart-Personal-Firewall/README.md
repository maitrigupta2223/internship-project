# 🔥 Smart Personal Firewall Dashboard

## 📌 Overview
A GUI-based personal firewall developed using Python and Scapy that performs live packet monitoring, rule-based filtering, and suspicious activity detection.

## 🚀 Features
- Live packet monitoring
- IP blocking
- Port blocking
- Suspicious activity detection
- Color-coded dashboard
- Logging system
- Export logs
- Firewall ON/OFF toggle

## 🛠 Technologies Used
- Python 3
- Scapy
- Tkinter
- JSON
- Logging module
- Kali Linux

## ▶️ How to Run
```bash
source venv/bin/activate
sudo venv/bin/python firewall.py
